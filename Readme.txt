# 📱 Smart Todo App

> 🚀 A modern mobile-first Todo App with Dashboard, Dark Mode & Smart Task Management

---

## ✨ Features

- 👋 Personalized welcome screen (stores username)
- 📊 Real-time dashboard (Total, Pending, Completed, High Priority)
- 📈 Animated progress bar with smooth transitions
- 🔍 Search tasks instantly
- 🎯 Filter tasks (All / Pending / Completed)
- 📝 Add, Edit & Delete tasks
- 🏷 Task priorities (Low, Medium, High)
- 📅 Due date support
- 🌙 Dark mode toggle
- ⚡ Smooth animations & micro-interactions
- 📱 Mobile-first responsive design
- 💾 Data persistence using localStorage

## 🛠 Tech Stack

- ⚛️ React.js (Hooks)
- 🎨 CSS3 (Custom UI + Animations)
- 💾 LocalStorage (Data persistence)
- 📱 Capacitor (Mobile support)

## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/your-repo.git

# Go to project
cd todo-app

# Install dependencies
npm install

# Run project
npm start

📱 Run on Mobile

npm run build
npx cap add android
npx cap open android
