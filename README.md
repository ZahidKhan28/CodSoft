# 📱 Smart Todo App

A modern mobile-first Todo App built using React.

---

## ✨ Features

- Add, edit, delete tasks
- Task priorities (Low, Medium, High)
- Due dates
- Dashboard (Total, Pending, Completed)
- Dark mode
- Local storage support
